# Dark Mode Toggle (Modern UI + GSAP)

This project is a modern **Dark Mode Toggle web application** built using **HTML, CSS, and Vanilla JavaScript** with **GSAP animations**. The app allows users to switch between **light mode and dark mode** with smooth animated transitions and a clean user interface.

## Features
- Modern and minimal **user interface**
- **Light Mode and Dark Mode** theme switching
- **GSAP animations** for smooth transitions
- Theme preference saved using **localStorage**
- Automatically loads the previously selected theme on page refresh
- **Hover effects and micro-interactions**

## Technologies Used
- HTML
- CSS
- JavaScript
- GSAP (GreenSock Animation Platform)
- LocalStorage API
